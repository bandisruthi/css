  • Style: The whole CSS rule, e.g., p{ color: blue; font-size: 18px;}.
  • Property: The visual aspect you want to change, e.g., color or font-size.
  • Value: The exact setting for that property, e.g., blue or 18px.

3 Ways to Apply CSS

1. Inline CSS

Style is written inside the html element using the style attribute. Good for quick tests but not recommended for big projects.

2. Internal (Embedded) CSS

Written inside a style tag in the head section of the html section of the html file. Useful for single-page websites.

3. External CSS

Stored in a separate css file and linked to html using a link tag in the head.Best practice for larger sites because one file can style multiple pages.
